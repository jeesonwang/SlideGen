# SlideGen

## 说明
Python: 3.11.8

### slidegen 文件夹结构

```
slidegen/
├── common/
    存放通用模块和工具函数。
├── config/
    存放配置文件。
├── contexts/
    存放上下文相关代码。
├── controller/
    存放控制器代码。
├── core/
    存放核心功能代码。
├── engine/
    存放引擎相关代码。
├── exception/
    存放异常处理相关代码。
├── middleware/
    存放中间件代码。
├── model/
    存放模型代码。
├── server.py
├── util/
    存放实用工具函数。
└── view/
    存放视图相关代码。
```
